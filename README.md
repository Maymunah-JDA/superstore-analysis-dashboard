# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes Superstore sales data to understand revenue, profit, customer behavior, and shipping performance. The dashboard provides insights to support data-driven business decisions.

## Objectives
- Analyze total sales, profit, and order volume
- Identify high-performing customer segments and regions
- Evaluate the impact of shipping modes and discounts on profitability

## Key Insights
- Standard Class shipping generates the highest sales and profit
- Consumer segment contributes the most revenue
- Certain regions outperform others in profitability
- Discounts significantly impact profit margins

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Slicers
- Data Cleaning and Analysis

## Dashboard Features
- Interactive filters by Segment, Region, Category, and Year
- KPI cards for Sales, Profit, Orders, Quantity, and Discount
- Visual analysis by Ship Mode, Segment, and Region
